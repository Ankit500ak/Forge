## Features Explained

### Stat Points
- Earned through fitness activities
- Used to redeem legendary equipment
- Contribute to rank progression
- Display in top-right of most pages

### Rank Tiers (Low to High)
- **F Rank**: 0-2,999 points (Starting)
- **E Rank**: 3,000-7,999 points (Beginner)
- **D Rank**: 8,000-14,999 points (Developing)
- **C Rank**: 15,000-24,999 points (Intermediate)
- **B Rank**: 25,000-34,999 points (Advanced)
- **A Rank**: 35,000-44,999 points (Elite)
- **S Rank**: 45,000+ points (Master)

### Charts Explained

**Radar Chart** (Stats Page)
- Shows all four stat categories at once
- Inner area = your performance
- Larger area = higher stats
- Compare across dimensions

**Line Charts** (Dashboard & Stats)
- Track progress over time
- Weekly XP growth
- Monthly stat progression
- Spot trends easily

**Bar Charts** (Dashboard & Stats)
- Compare values side-by-side
- Current vs goal comparisons
- Category-by-category breakdowns
- Hover for exact values

## Troubleshooting

**Can't login?**
- Make sure you've created an account first
- Check email and password are correct
- Try the demo account instead

**Stats not showing?**
- Refresh the page
- Make sure you're logged in
- Check that navigation loaded properly

**Charts not displaying?**
- Clear browser cache
- Refresh the page
- Try a different browser
- Check screen size (some might be too small for charts)

**Mobile nav not working?**
- Scroll to bottom of page
- Tap the icons clearly
- Make sure you're on mobile view
